# Reference: Spacing & Layout

Шкала отступов в SDDS. Базовый шаг — **4px**. Все значения кратны 4.

> SDDS не использует отдельные spacing-переменные (CSS-переменные `--spacing/N` отсутствуют). Компоненты применяют значения как hardcoded px. Шкала ниже — справочник допустимых значений.

---

## Шкала

| px | T-shirt | Применение |
|---|---|---|
| 4 | — | Внутренние отступы мелких элементов, gap внутри Badge/Chip |
| 8 | — | Между иконкой и лейблом внутри компонента |
| 12 | XS | Padding XS-компонентов |
| 16 | S | Padding S-компонентов |
| 20 | M | Padding M-компонентов |
| 24 | L | Padding L-компонентов |
| 28 | XL | Padding XL-компонентов |
| 32 | — | Между крупными блоками, gap между секциями формы |
| 40 | — | Секционный отступ, разделение смысловых групп |

Каждая ступень = +4px к предыдущей.

---

## Внутренние padding'и

### Кнопки и поля ввода (горизонтальный padding)

| Размер | Высота | Padding H |
|---|---|---|
| XS | 32px | 12px |
| S | 40px | 16px |
| M | 48px | 20px |
| L | 56px | 24px |
| XL | 64px | 28px |

Padding кнопок и полей одной ступени **совпадает** — это даёт визуальное выравнивание, когда они стоят в одной строке.

---

## Расстояния между компонентами

| Контекст | Gap |
|---|---|
| Иконка и лейбл внутри компонента | 8px |
| Поля в inline-группе (TextField + Button) | 8px |
| Поля в вертикальной форме | 16px |
| Группы полей между собой | 24px |
| Секции формы / смысловые блоки | 32–40px |
| Отдельные карточки / панели | 16–24px |

---

## Правила

- Используйте только значения из шкалы
- Не уменьшайте gap между полями формы ниже 16px (элементы сливаются)
- Один размер отступа на контекст: не смешивайте 16px и 24px между полями одной формы
- Сохраняйте вертикальный ритм — все отступы кратны 4px
