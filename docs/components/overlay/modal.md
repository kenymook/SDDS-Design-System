# Modal

> **Category** · Overlay
> **Figma page** · `14049:14588`

Диалоговое окно, блокирующее взаимодействие с основным интерфейсом.

**Use** — для подтверждения действий, форм, деталей объекта.

**Slots:** `header` · `body` · `footer`

**Keyboard:** `Escape` — закрыть · `Tab` — навигация внутри · фокус-ловушка активна пока открыт.

**Accessibility:** `role="dialog"` · `aria-modal="true"` · `aria-labelledby` на заголовок.
