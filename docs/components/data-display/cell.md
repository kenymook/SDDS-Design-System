## 1. Key Principles of Use

### What it is

Cell — строка-ячейка для списков и таблиц. Стандартизированный элемент строки с иерархией слотов.

### When to use

**Use** — для строк в списках, пунктов меню настроек, строк таблицы с произвольным контентом.
