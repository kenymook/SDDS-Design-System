## 1. Key Principles of Use

### What it is

Loader — индикатор загрузки с определённым прогрессом (в отличие от Spinner). Отображает долю выполненного.

### When to use

**Use** — когда известна доля выполненного: загрузка файла, инициализация с этапами.

**Don't use** — когда прогресс неизвестен (используйте **Spinner**).
