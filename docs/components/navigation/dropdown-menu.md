# DropdownMenu

> **Category** · Navigation
> **Figma page** · `12023:497773`

Выпадающее меню действий. Открывается по клику на триггер.

**Keyboard:** `Escape` — закрыть · `Arrow keys` — навигация · `Enter`/`Space` — выбор.

**Accessibility:** `role="menu"` · `role="menuitem"` · `aria-expanded` на триггере.
