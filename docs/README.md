# Документация SDDS

Каноническая база знаний о дизайн-системе SDDS.

## Разделы

- [Getting Started](getting-started/) — последовательное знакомство с системой по ролям.
- [Foundations](foundations/) — базовые модели: introducing, glossary, токены (theming), размеры (sizes), скругления (corner-radius), состояния (states), словарь пропов (props-vocabulary), библиотека компонентов (components), доступность (accessibility).
- [Concepts](concepts/) — объяснение архитектурных и продуктовых решений.
- [Guides](guides/) — практические рецепты для конкретных задач.
- [Reference](reference/) — справочные таблицы и точные определения.
- [Components](components/) — спецификации компонентов по категориям.
- [Assets](assets/) — диаграммы и иллюстрации документации.
