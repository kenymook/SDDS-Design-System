# Архив документации

Здесь хранятся прежние версии страниц, оставленные для сверки после реорганизации.

- `legacy-root/` — документы, которые раньше находились в корне репозитория.
- `legacy-guides/` — руководства, объединённые с более полными каноническими страницами.

Архив не является источником истины. Новые документы и ссылки должны указывать на канонические разделы `docs/`.
