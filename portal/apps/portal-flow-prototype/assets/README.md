# Assets прототипа

Реальные скриншоты продуктов для слайдера «В реальных продуктах» на главной.

Положите файлы сюда (именно с такими именами):

- `giga.webp` — GigaChat (Web) — ✅ уже добавлен
- `homeos.webp` — HomeOS (Smart TV) — ✅ уже добавлен
- `sberdevices.jpg` — SberDevices (Web) — ✅ уже добавлен
- `finai.webm` — FinAI (Web) — ✅ уже добавлен (видео-луп)
- `platformai.webm` — PlatformAI (Web) — ✅ уже добавлен (видео-луп)
- `sberscan.webp` — SberScan (Mobile) — ✅ уже добавлен
- `showcase-netologiya.png` — Нетология (Web)
- `showcase-sberstrahovanie.png` — СберСтрахование (Web · Mobile)

Пока файла нет — на его месте показывается плейсхолдер с названием продукта. Форматы: PNG / JPG / WebP, а также видео-лупы WebM / MP4 (подхватываются автоматически по расширению, играют без звука по кругу). Медиа центрируется с обрезкой под пропорции блока.

Чтобы добавить новый продукт в слайдер: добавьте объект в `showcaseShots` в `app.js` и положите файл сюда.
