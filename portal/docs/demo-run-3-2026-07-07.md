## Ключевые наблюдения по новому

### Путь «главная → компоненты» (Марина)
CTA «Посмотреть компоненты» → хаб → плитка Select → спецификация с живым превью → крошка «Документация» → хаб. Полный круг без тупиков. «Главный пробел прогонов №1–2 закрыт: я вижу, что входит в систему, за два клика с первого экрана».

### Core principles (Марина + Олег)
Новый состав (White-label · Light и Dark из коробки · 4 контекста темы · Единые словари системы) с мини-визуализациями. Марина: «Раньше я эту секцию проматывала — четыре абзаца определений. Теперь визуалы делают мысль считываемой за секунду, а карточка про контексты — первое место на странице, где я узнала что-то, чего не ожидала». Олег: «Дубли исчезли — страница перестала повторяться».

### Повествование главной (все)
Последовательность заголовков читается как связный рассказ без заиканий; live demo подтверждает карточки принципов (Light/Dark и размеры переключаются руками через один экран после того, как о них заявили).
