# Настройки Design System

**Путь (route):** `design-system-settings` · **Часть:** DS Builder (после входа) · **Прототип:** Design System → настройки (только Owner)

## 1. Зачем экран
Управление дизайн-системой как сущностью: название, обложка, перенос в другой проект, удаление. Административные действия, отдельные от работы с темами.

## 2. Кто видит и что может
| Роль | Что может |
|---|---|
| Viewer / Editor | Нет доступа — экран «нет доступа» |
| Owner | Полный доступ |

## 3. Из чего состоит
1. **Заголовок** — «Настройки Design System» + название, кнопка «Готово».
2. **Основное** — поле «Название».
3. **Обложка** — оформление.
4. **Перенести в другой проект** — выбор целевого проекта (только где ты Owner); темы, версии и история переезжают вместе.
5. **Удалить Design System** — опасная зона с вводом названия для подтверждения.

## 4. Состояния
| Состояние | Есть? | Что показываем |
|---|:--:|---|
| Обычное | ✓ | Формы настроек |
| Нет доступа (не Owner) | ✓ | Экран «нет доступа» |
| Нельзя перенести | ✓ | Выбор и кнопка неактивны + пояснение «нет другого проекта, где вы Owner» |
| Проверка / ошибка | ✓ | Ошибки в полях и над формами |
| Подтверждение удаления | ✓ | Удаление после ввода точного названия |

## 5. Действия и переходы
| Действие | Когда | Что происходит |
|---|---|---|
| Сохранить название | заполнено | Обновляет систему |
| Задать обложку | всегда | Меняет обложку |
| Перенести | есть целевой проект | Переносит систему со всем содержимым |
| Удалить | введено название | Удаляет систему и её темы/версии |
| Готово | всегда | Назад к обзору Design System |

## 6. Как сюда попадают и куда уходят
- **Входы:** обзор Design System (кнопка настроек).
- **Выходы:** обзор Design System.

## 7. Тексты
Понятно предупреждаем, что перенос тянет за собой темы, версии и историю; удаление защищено вводом названия.

## 8. Телефон
Формы в одну колонку.

## 9. Доступность
Опасные действия требуют осознанного ввода; недоступный перенос объяснён текстом.

## 10. Что отрисовать (чек-лист макетов)
- [ ] Обычное (формы)
- [ ] Нет доступа (не Owner)
- [ ] Перенос недоступен
- [ ] Опасная зона: удаление
- [ ] Телефон

## 11. Что ещё не решено
