# Обзор проекта

**Путь (route):** `project` · **Часть:** DS Builder (после входа) · **Прототип:** выбрать проект в панели

## 1. Зачем экран
Главная страница проекта: список его Design System. Отсюда заходят в систему, создают новую (Owner) и попадают в настройки проекта (Owner).

## 2. Кто видит и что может
| Роль | Что может |
|---|---|
| Viewer / Editor | Смотрят список Design System, заходят внутрь |
| Owner | + создать Design System, + настройки проекта, + меню у карточек |

## 3. Из чего состоит
1. **Шапка коллекции** — название проекта, заголовок «Design Systems», фильтры (все / свои / добавленные и сортировка), кнопки «создать» и «настройки» (у Owner).
2. **Сетка карточек** — каждая Design System: обложка, название, статус и число тем; меню «⋯» (у Owner).

## 4. Состояния
| Состояние | Есть? | Что показываем |
|---|:--:|---|
| Обычное | ✓ | Сетка Design System |
| Пусто (нет систем) | ✓ | «В проекте пока нет Design Systems»; у Owner — намёк создать |
| Пусто по фильтру | ✓ | «Ничего не найдено» + «Показать все» |
| Только для просмотра | ✓ | Нет кнопок создания и настроек |

## 5. Действия и переходы
| Действие | Когда | Что происходит |
|---|---|---|
| Открыть Design System | всегда | → Обзор Design System |
| Создать Design System | Owner | → Создать Design System |
| Настройки проекта | Owner | → Настройки проекта |
| Фильтр / сортировка | всегда | Меняет список |
| Меню карточки | Owner | Переименовать / перенести / удалить |

## 6. Как сюда попадают и куда уходят
- **Входы:** панель проектов, после входа, уведомление.
- **Выходы:** обзор Design System, создание, настройки проекта.

## 7. Тексты
Статус и число тем на карточке — коротко. Пустые состояния объясняют, что делать и кому это доступно.

## 8. Телефон
Сетка в одну колонку; фильтры сворачиваются.

## 9. Доступность
Карточки доступны с клавиатуры (переход по Enter); меню открывается с клавиатуры.

## 10. Что отрисовать (чек-лист макетов)
- [ ] Обычное (сетка)
- [ ] Пусто (нет систем) — для Owner и не-Owner
- [ ] Пусто по фильтру
- [ ] Меню карточки
- [ ] Телефон

## 11. Что ещё не решено
Нет открытых вопросов сверх общих.
