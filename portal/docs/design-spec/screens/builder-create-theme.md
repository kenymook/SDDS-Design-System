# Создать тему

**Путь (route):** `create-theme` · **Часть:** DS Builder (после входа) · **Прототип:** Design System → создать тему (Owner или Editor)

## 1. Зачем экран
Завести новую тему внутри дизайн-системы и выбрать стартовую палитру — она задаст начальные цвета. Потом значения правятся уже внутри темы.

## 2. Кто видит и что может
| Роль | Что может |
|---|---|
| Viewer | Нет доступа — экран «нет доступа» |
| Owner / Editor | Создаёт тему |

## 3. Из чего состоит
1. **Заголовок** — «Создать тему» (с проектом и системой в крошках).
2. **Название** (обязательно).
3. **Стартовая палитра** — выбор основы: Base (системная), Malachite (малахитовый бренд-акцент), B2B (строгая B2B-основа), Custom (своя брендовая палитра). У Custom — свои пресеты (нейтральный / брендовый / тёплый) и возможность задать значения; они применятся к светлому и тёмному режимам.
4. **Действия** — отмена и создать.

## 4. Состояния
| Состояние | Есть? | Что показываем |
|---|:--:|---|
| Обычное | ✓ | Форма с выбранной палитрой по умолчанию |
| Нет доступа (Viewer) | ✓ | Экран «нет доступа» |
| Выбран Custom | ✓ | Дополнительный блок пресетов и значений |
| Проверка | ✓ | Ошибка пустого названия |

## 5. Действия и переходы
| Действие | Когда | Что происходит |
|---|---|---|
| Выбрать палитру | всегда | Задаёт начальные цвета |
| Выбрать пресет Custom | палитра Custom | Меняет основу нейтральный/брендовый/тёплый |
| Создать | заполнено | Создаёт тему в статусе «черновик», открывает редактор |
| Отмена | всегда | Назад к Design System |

## 6. Как сюда попадают и куда уходят
- **Входы:** обзор Design System (кнопка «создать тему»).
- **Выходы:** новая тема (редактор).

## 7. Тексты
Поясняем, что палитра — это только старт: значения потом редактируются внутри темы, и что светлый и тёмный режимы создаются сразу.

## 8. Телефон
Варианты палитры — стопкой; форма в одну колонку.

## 9. Доступность
Выбор палитры — радиокнопками, доступными с клавиатуры; у образцов цвета есть текстовые подписи.

## 10. Что отрисовать (чек-лист макетов)
- [ ] Base выбрана
- [ ] Malachite выбрана
- [ ] B2B выбрана
- [ ] Custom + пресеты (нейтральный/брендовый/тёплый)
- [ ] Нет доступа (Viewer)
- [ ] Проверка
- [ ] Телефон

## 11. Что ещё не решено
Нет открытых вопросов сверх общих.
