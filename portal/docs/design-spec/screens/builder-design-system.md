# Обзор Design System

**Путь (route):** `design-system` · **Часть:** DS Builder (после входа) · **Прототип:** открыть Design System в проекте

## 1. Зачем экран
Страница дизайн-системы: список её тем. Отсюда заходят в тему на редактирование, создают новую (Owner или Editor) и попадают в настройки системы (Owner).

## 2. Кто видит и что может
| Роль | Что может |
|---|---|
| Viewer | Смотрит темы, заходит внутрь |
| Editor | + создать тему, + меню у карточек |
| Owner | + настройки Design System |

## 3. Из чего состоит
1. **Шапка коллекции** — название системы, кнопка «назад» к проекту, заголовок «Themes», фильтры, «создать тему» (Owner/Editor) и «настройки» (Owner).
2. **Сетка карточек** — каждая тема: обложка, название, статус (черновик / опубликована) и версия; меню «⋯» (не для Viewer).

## 4. Состояния
| Состояние | Есть? | Что показываем |
|---|:--:|---|
| Обычное | ✓ | Сетка тем |
| Пусто (нет тем) | ✓ | «В Design System пока нет Themes» |
| Пусто по фильтру | ✓ | «Ничего не найдено» + «Показать все» |
| Только для просмотра | ✓ | Нет создания и меню карточек |

## 5. Действия и переходы
| Действие | Когда | Что происходит |
|---|---|---|
| Открыть тему | всегда | → Редактор темы |
| Создать тему | Owner/Editor | → Создать тему |
| Настройки Design System | Owner | → Настройки Design System |
| Назад к проекту | всегда | → Обзор проекта |
| Фильтр / сортировка | всегда | Меняет список |
| Меню карточки | не Viewer | Переименовать / перенести / удалить |

## 6. Как сюда попадают и куда уходят
- **Входы:** обзор проекта, уведомление.
- **Выходы:** редактор темы, создание темы, настройки системы, назад к проекту.

## 7. Тексты
На карточке — статус «черновик / опубликована» и версия. Пустые состояния подсказывают, что делать.

## 8. Телефон
Сетка в одну колонку; фильтры сворачиваются.

## 9. Доступность
Карточки доступны с клавиатуры; статус передаётся текстом, не только цветом.

## 10. Что отрисовать (чек-лист макетов)
- [ ] Обычное (сетка тем)
- [ ] Пусто (нет тем)
- [ ] Пусто по фильтру
- [ ] Меню карточки
- [ ] Viewer (без создания)
- [ ] Телефон

## 11. Что ещё не решено
